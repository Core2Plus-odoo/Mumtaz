ZATCA (Saudi e-invoicing)
Set VAT number and seller name first, then click "Generate CSR" to create the keypair you'll submit to the Fatoora portal.
Onboarding flow: 1. Set ZATCA_VAT_NUMBER + ZATCA_SELLER_NAME + ZATCA_ENV above and Save. 2. Click Generate CSR below — saves the private key into the DB. 3. Get an OTP from Fatoora portal. 4. POST the CSR + OTP to /api/zatca/onboard — returns a CSID we save automatically.
