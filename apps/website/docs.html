ZATCA Phase 2 Integration Updated 2026

All Mumtaz plans include built-in ZATCA Phase 2 clearance integration. The following steps walk you through onboarding your CCSID and configuring automatic clearance.

Prerequisites

You will need: your Saudi VAT registration number (15 digits), your ZATCA portal credentials, and a valid CCSID certificate. Mumtaz handles the CSR generation automatically.

Configuration

Navigate to Settings → Compliance → E-Invoicing. Enter your VAT number and click Generate CSR. Mumtaz will create a cryptographic signing key pair and display the CSR for submission to FATOORAH.

# ZATCA e-invoice structure (simplified)
{
  "invoiceTypeCode": "388",
  "documentCurrencyCode": "SAR",
  "taxCurrencyCode": "SAR",
  "invoiceLines": [...],
  "taxTotal": { "taxAmount": "15.00" }
}

Clearance flow

Standard invoices (B2B above SAR 1,000) go through clearance mode — Mumtaz sends the signed XML to FATOORAH before delivery to the buyer and stores the cleared XML with the ZATCA stamp. Below-threshold and B2C invoices go through reporting mode automatically.

API Reference Beta

The Mumtaz REST API lets you read and write ERP data from external systems. All endpoints require a Bearer token issued through the auth endpoint.

Authentication

POST /api/v1/auth/token
Content-Type: application/json

{
  "email": "admin@yourcompany.com",
  "password": "••••••••"
}

# Response
{
  "access_token": "eyJ...",
  "token_type": "Bearer",
  "odoo_db": "mt_yourcompany_a1b2c3"
}

Common endpoints

Once authenticated, pass Authorization: Bearer {token} on every request.

GET  /api/v1/invoices          # List invoices
POST /api/v1/invoices          # Create invoice
GET  /api/v1/invoices/{id}     # Get invoice details
POST /api/v1/invoices/{id}/pay # Record payment

GET  /api/v1/customers         # List customers
GET  /api/v1/products          # List products
GET  /api/v1/reports/pl        # Profit & loss summary

Rate limits

Starter: 60 requests/minute. Professional: 300 requests/minute. Enterprise: custom. Rate limit headers are included in every response: X-RateLimit-Remaining and X-RateLimit-Reset.

ZAKI AI — Getting Started

ZAKI is an AI agent embedded directly in your Mumtaz workspace. It has read access to all your ERP data and can answer questions in plain language, run reports, and flag anomalies.

How to access ZAKI

Click the ZAKI button in the top-right of any ERP screen, or press Cmd+K (Mac) / Ctrl+K (Windows) from anywhere in the platform.

Example prompts

"What's my cash position today?"
ZAKI pulls bank balances, outstanding receivables, and payables due this week.
"Which customers are overdue by more than 30 days?"
Returns an aged receivables breakdown with contact details.
"Show me gross margin by product line this quarter."
Generates a formatted table with variance vs. prior quarter.
"Draft a payment reminder for invoice INV-1042."
Writes a professional email in Arabic or English, ready to send.
