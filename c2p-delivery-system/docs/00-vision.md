# Vision

C2P Agency OS — the operating system for AI-run Odoo delivery agencies.

Build it to run C2P Consultants (customer #0), prove it on our own pipeline, then
license the same codebase to other Odoo partners as a C2P Ventures product.

- Lead generation → branded proposals → approval → project delivery → ongoing.
- Specialist Odoo agents (functional + technical) that remember every client.
- A human owner approves only money, reputation, and production-code decisions.
- Odoo is the system of record; agents augment, never rebuild standard.

See 02-master-build-brief.md for the full system and 03-productization-and-
organization.md for the product and repo organization.
